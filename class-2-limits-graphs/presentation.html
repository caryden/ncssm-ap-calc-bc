For Every Limit Problem, Ask:

1 What is \(\lim_{x \to a^-} f(x)\)? (left-hand limit)
2 What is \(\lim_{x \to a^+} f(x)\)? (right-hand limit)
3 Does \(\lim_{x \to a} f(x)\) exist? (two-sided limit)
4 What is f(a)? (function value)

The Two-Sided Limit Exists When...

\(\lim_{x \to a} f(x) = L\) exists if and only if
\(\lim_{x \to a^-} f(x) = \lim_{x \to a^+} f(x) = L\)

Limit vs. f(a): They Can Differ!

The Limit

\(\lim_{x \to a} f(x)\)

Where the curve is heading

Follow the path, not the point

f(a)

The actual value

Where the dot is located

May be different, or undefined

The filled dot shows f(a). The limit is about the curve's direction.

What the Graph Tells You

Solid dot = f(a) exists
Open circle = f(a) undefined at this height

The curve's direction tells you the limit; dots tell you about f(a)

Case 1: Continuous at x = a

\(\lim_{x \to a} f(x) = f(a)\)

Everything agrees. Limit equals function value.

Case 2: Hole in the Graph

Limit exists, but \(\lim_{x \to a} f(x) \neq f(a)\)

The limit exists, but f(a) is somewhere else.

Case 3: Limit Exists, f(a) Undefined

Limit exists, f(a) is undefined

The curve approaches a value, but there's no actual value at a.

Case 4: Jump Discontinuity

\(\lim_{x \to a^-} f(x) \neq \lim_{x \to a^+} f(x)\)

Left and right limits disagree. Two-sided limit DNE.

Case 5: Unbounded Behavior

Function approaches \(\pm\infty\)

When f(x) grows without bound, the limit does not exist (as a finite number).

Case 6: Wild Oscillation

\(f(x) = \sin(1/x)\) oscillates infinitely near x = 0

The function oscillates infinitely fast—no single value is approached.

Avoid These Errors

  • Confusing the dot (f(a)) with the limit
  • Only checking one direction of approach
  • Missing open circles on the graph
  • Reading the wrong y-value from the curve

Key Takeaways

  • Trace the curve from both sides
  • The limit is about direction, not destination
  • Both one-sided limits must agree for the two-sided limit to exist
  • f(a) and the limit are independent concepts
