Increasing, Decreasing, and Local Extrema

If you know the sign of \( f'(x) \), what do you know about \( f(x) \)?

Warm-Up

For \( f(x) = x^3 - 3x^2 + 2 \):

  1. Find all critical points
  2. Determine the sign of \( f'(x) \) on each interval
  3. Where is \( f \) increasing? Decreasing?

Increasing and Decreasing Functions

Definitions

\( f \) is increasing on an interval if for any \( x_1 < x_2 \): \( f(x_1) < f(x_2) \)

\( f \) is decreasing on an interval if for any \( x_1 < x_2 \): \( f(x_1) > f(x_2) \)

Increasing = going up as you move right
Decreasing = going down as you move right

The Derivative Tells Us Everything

Increasing/Decreasing Test

If \( f'(x) > 0 \) on an interval, then \( f \) is increasing on that interval.

If \( f'(x) < 0 \) on an interval, then \( f \) is decreasing on that interval.

Why? By MVT, if \( f'(x) > 0 \), then any secant line has positive slope, so the function rises!

Visualizing Increasing/Decreasing

\( f'(x) > 0 \): Increasing
\( f'(x) < 0 \): Decreasing

Sign Charts for \( f'(x) \)

How to Build a Sign Chart

  1. Find all critical points (where \( f' = 0 \) or DNE)
  2. Plot critical points on a number line
  3. Test a value in each interval
  4. Mark each interval as + or -

Example: Sign Chart for \( f(x) = x^3 - 3x \)

1 \( f'(x) = 3x^2 - 3 = 3(x-1)(x+1) \)
2 Critical points: \( x = -1 \) and \( x = 1 \)
3 Test: \( f'(-2) = 9 > 0 \), \( f'(0) = -3 < 0 \), \( f'(2) = 9 > 0 \)
4 Increasing on \( (-\infty, -1) \) and \( (1, \infty) \); Decreasing on \( (-1, 1) \)

The First Derivative Test

First Derivative Test for Local Extrema

Suppose \( c \) is a critical point of a continuous function \( f \):

Local Maximum

If \( f' \) changes from + to - at \( c \), then \( f(c) \) is a local maximum.

Local Minimum

If \( f' \) changes from - to + at \( c \), then \( f(c) \) is a local minimum.

Neither

If \( f' \) does not change sign at \( c \), then \( f(c) \) is neither.

First Derivative Test in Action

Local Max (+to-)
Local Min (-to+)

If you're going up and then going down,
you must have reached a peak.

The sign change of \( f' \) tells us the direction change of \( f \)!

Example: Complete Analysis

Find all local extrema of \( f(x) = x^4 - 4x^3 \) using the First Derivative Test.

1 \( f'(x) = 4x^3 - 12x^2 = 4x^2(x - 3) \)
2 Critical points: \( x = 0 \) and \( x = 3 \)
3 Sign chart: \( f'(-1) < 0 \), \( f'(1) < 0 \), \( f'(4) > 0 \)
4 At \( x = 0 \): no sign change (neither). At \( x = 3 \): - to + (local min)

Exploring \( f(x) = x^4 - 4x^3 \)

x = 0: Neither (- to -)
x = 3: Local Min (- to +)

Example: Rational Function

Find intervals of increase/decrease for \( f(x) = \frac{x^2}{x-1} \).

1 \( f'(x) = \frac{2x(x-1) - x^2}{(x-1)^2} = \frac{x^2 - 2x}{(x-1)^2} = \frac{x(x-2)}{(x-1)^2} \)
2 Critical: \( x = 0, 2 \). Not in domain: \( x = 1 \)
3 Sign chart with four intervals separated by 0, 1, 2
4 Inc: \( (-\infty, 0) \cup (2, \infty) \), Dec: \( (0,1) \cup (1,2) \)

Common Mistakes to Avoid

Forgetting DNE

Critical points include where \( f' \) doesn't exist!

Ignoring Domain

Discontinuities break up intervals too.

Wrong Conclusion

\( f'(c) = 0 \) alone doesn't guarantee extremum.

What \( f' \) Tells Us About \( f \)

Key Observations

  • \( f' > 0 \) means \( f \) is increasing
  • \( f' < 0 \) means \( f \) is decreasing
  • \( f' = 0 \) means \( f \) has horizontal tangent
  • \( f' \) crosses x-axis means potential extremum of \( f \)
