Formal Definition of Concavity

Definition

\( f \) is concave up on an interval if \( f' \) is increasing on that interval.

\( f \) is concave down on an interval if \( f' \) is decreasing on that interval.

Intuition: Concavity describes how the slope itself is changing!

Concavity and the Second Derivative

Concavity Test

If \( f''(x) > 0 \) on an interval, then \( f \) is concave up.

If \( f''(x) < 0 \) on an interval, then \( f \) is concave down.

Memory Aid

\( f'' > 0 \): smile \( \cup \)      \( f'' < 0 \): frown \( \cap \)

Visualizing Concavity

Concave Up (\( f'' > 0 \))
Concave Down (\( f'' < 0 \))

Inflection Points

Definition: Inflection Point

An inflection point is where the concavity changes.

At an inflection point, \( f''(c) = 0 \) or \( f''(c) \) does not exist.

Caution: \( f''(c) = 0 \) alone is not enough! The sign must actually change.

Finding Inflection Points

Inflection at x = 0: concave down to concave up

Example: Concavity Analysis

Find intervals of concavity and inflection points for \( f(x) = x^3 - 6x^2 + 9x + 1 \).

1 \( f'(x) = 3x^2 - 12x + 9 \)
2 \( f''(x) = 6x - 12 = 6(x - 2) \)
3 \( f''(x) = 0 \) when \( x = 2 \)
4 Concave down on \( (-\infty, 2) \), concave up on \( (2, \infty) \)
5 Inflection point at \( x = 2 \) (sign changes!)

The Second Derivative Test

Second Derivative Test for Local Extrema

Suppose \( f'(c) = 0 \):

Local Maximum

If \( f''(c) < 0 \), then \( f(c) \) is a local maximum.

Local Minimum

If \( f''(c) > 0 \), then \( f(c) \) is a local minimum.

Inconclusive

If \( f''(c) = 0 \), the test fails. Use FDT instead.

Why the Second Derivative Test Works

At a critical point where \( f' = 0 \):

\( f'' < 0 \): curve bends down = local max (peak)

\( f'' > 0 \): curve bends up = local min (valley)

Example: Using the Second Derivative Test

Find and classify all local extrema of \( f(x) = x^4 - 8x^2 \).

1 \( f'(x) = 4x^3 - 16x = 4x(x^2 - 4) = 4x(x-2)(x+2) \)
2 Critical points: \( x = -2, 0, 2 \)
3 \( f''(x) = 12x^2 - 16 \)
4 \( f''(-2) = 32 > 0 \) (min), \( f''(0) = -16 < 0 \) (max), \( f''(2) = 32 > 0 \) (min)

What Each Derivative Tells Us

Summary of Connections

  • \( f' > 0 \): \( f \) increasing  |  \( f' < 0 \): \( f \) decreasing
  • \( f' = 0 \): possible extremum of \( f \)
  • \( f'' > 0 \): \( f \) concave up, \( f' \) increasing
  • \( f'' < 0 \): \( f \) concave down, \( f' \) decreasing
  • \( f'' = 0 \) (with sign change): inflection point of \( f \), extremum of \( f' \)

Complete Analysis

Analyze \( f(x) = x^4 - 4x^3 + 10 \) completely.

1 \( f'(x) = 4x^3 - 12x^2 = 4x^2(x - 3) \). Critical: \( x = 0, 3 \)
2 \( f''(x) = 12x^2 - 24x = 12x(x - 2) \). Zero at \( x = 0, 2 \)
3 SDT: \( f''(3) = 36 > 0 \), so \( x = 3 \) is local min. \( f''(0) = 0 \) is inconclusive.
4 Inflection points at \( x = 0 \) and \( x = 2 \) (concavity changes)

Key Takeaways

Concavity: \( f'' > 0 \) means concave up; \( f'' < 0 \) means concave down

Inflection: Where concavity changes (and \( f'' = 0 \) or DNE)

SDT: At critical point, \( f'' < 0 \) means max; \( f'' > 0 \) means min
