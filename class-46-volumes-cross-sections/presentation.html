From 2D Area to 3D Volume

Just as we found area by summing thin rectangles...

We find volume by summing thin slices!

\[\text{Area} = \int_a^b f(x) \, dx \quad \longrightarrow \quad \text{Volume} = \int_a^b A(x) \, dx\]

The Cross Section Method

Volume by Cross Sections

\[V = \int_a^b A(x) \, dx\]

where \(A(x)\) is the cross-sectional area at position \(x\)

Think of it as:

Slicing a loaf of bread and adding up all the slice areas!

Setting Up Cross-Section Problems

1
Identify the base region

Sketch the 2D region in the xy-plane

2
Determine the base width

Find the distance across the base at position x (or y)

3
Write A(x) in terms of the base width

Use the appropriate area formula for the cross-section shape

4
Integrate!

\(V = \int_a^b A(x)\,dx\)

Example 1: Square Cross Sections

The base of a solid is the region bounded by \(y = \sqrt{x}\), \(y = 0\), and \(x = 4\).

Cross sections perpendicular to the x-axis are squares. Find the volume.

Example 1: Solution

Step 1: Base width at x

Width \(= \sqrt{x} - 0 = \sqrt{x}\)

Step 2: Cross-section area

Square: \(A(x) = (\sqrt{x})^2 = x\)

Step 3: Set up integral

\[V = \int_0^4 x \, dx\]

Step 4: Evaluate

\[V = \left[\frac{x^2}{2}\right]_0^4 = 8\]

Explore: Different Cross Sections

Example 2: Semicircular Cross Sections

The base is bounded by \(y = \sin x\) and \(y = 0\) from \(x = 0\) to \(x = \pi\).

Cross sections perpendicular to the x-axis are semicircles. Find the volume.

Base width: \(\sin x\) is the diameter, so \(r = \frac{\sin x}{2}\)

Area: \(A(x) = \frac{1}{2}\pi r^2 = \frac{\pi}{2} \cdot \frac{\sin^2 x}{4} = \frac{\pi \sin^2 x}{8}\)

Volume:

\[V = \int_0^{\pi} \frac{\pi \sin^2 x}{8} dx = \frac{\pi}{8} \cdot \frac{\pi}{2} = \frac{\pi^2}{16}\]

Cross Sections Perpendicular to the y-axis

Integrating with respect to y

\[V = \int_c^d A(y) \, dy\]

When to use dy:

  • When cross sections are perpendicular to the y-axis
  • When the base width is naturally expressed in terms of y
  • When boundaries are functions of y

Example 3: Using dy

The base is bounded by \(x = y^2\) and \(x = 1\).

Cross sections perpendicular to the y-axis are squares. Find the volume.

Base width: \(1 - y^2\) (right minus left)

Area: \(A(y) = (1 - y^2)^2\)

Limits: \(y^2 = 1 \Rightarrow y = \pm 1\)

\[V = \int_{-1}^{1} (1-y^2)^2 dy = \int_{-1}^{1} (1 - 2y^2 + y^4) dy = \frac{16}{15}\]

Example 4: Triangular Base

The base is a triangle with vertices (0,0), (2,0), and (0,2).

Cross sections perpendicular to the x-axis are equilateral triangles.

Hypotenuse equation: \(y = 2 - x\)

Base width at x: \(2 - x\)

Area: \(A(x) = \frac{\sqrt{3}}{4}(2-x)^2\)

\[V = \int_0^{2} \frac{\sqrt{3}}{4}(2-x)^2 dx = \frac{\sqrt{3}}{4} \cdot \frac{8}{3} = \frac{2\sqrt{3}}{3}\]

Circular Base

The base is a circle: \(x^2 + y^2 = r^2\). Cross sections are squares.

At position x: \(y = \pm\sqrt{r^2 - x^2}\)

Base width: \(2\sqrt{r^2 - x^2}\)

Area: \(A(x) = 4(r^2 - x^2)\)

\[V = \int_{-r}^{r} 4(r^2 - x^2) dx = 4\left[r^2 x - \frac{x^3}{3}\right]_{-r}^{r} = \frac{16r^3}{3}\]

Note: For a unit circle (r = 1), \(V = \frac{16}{3}\)

Practice Problems

  1. Base: \(y = x^2\) and \(y = 4\). Cross sections (perp. to y): squares. Find V.
  2. Base: \(y = \cos x\) and \(y = 0\) from \(x = -\frac{\pi}{2}\) to \(x = \frac{\pi}{2}\). Cross sections (perp. to x): squares. Find V.
  3. Base: triangle with vertices (0,0), (1,0), (0,1). Cross sections (perp. to x): semicircles. Find V.
  4. Base: \(y = e^{-x}\), \(y = 0\), \(x = 0\), \(x = 1\). Cross sections: isosceles right triangles with leg on the base. Find V.

Solutions

  1. Width = \(2\sqrt{y}\), \(A(y) = 4y\), \(V = \int_0^4 4y\,dy = 32\)
  2. \(A(x) = \cos^2 x\), \(V = \int_{-\pi/2}^{\pi/2} \cos^2 x\,dx = \frac{\pi}{2}\)
  3. Width = \(1-x\), \(r = \frac{1-x}{2}\), \(A(x) = \frac{\pi(1-x)^2}{8}\), \(V = \frac{\pi}{8} \cdot \frac{1}{3} = \frac{\pi}{24}\)
  4. \(A(x) = \frac{1}{2}e^{-2x}\), \(V = \int_0^1 \frac{1}{2}e^{-2x}dx = \frac{1}{4}(1 - e^{-2})\)
