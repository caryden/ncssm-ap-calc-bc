Unit 1: Limits and Continuity

The Squeeze Theorem

Limits by Comparison

Class 6 | February 12, 2026 | CED 1.8

A Limit We Can't Factor

\[\lim_{x \to 0} x^2 \sin\left(\frac{1}{x}\right)\]

Direct substitution gives 0 times undefined. We need a new approach.

If you're trapped between two things approaching the same place, you must go there too.

The Squeeze Theorem

Squeeze Theorem (Sandwich Theorem)

If \(g(x) \leq f(x) \leq h(x)\) for all x near a (except possibly at a), and

\[\lim_{x \to a} g(x) = \lim_{x \to a} h(x) = L\]

Then: \(\displaystyle\lim_{x \to a} f(x) = L\)

Visualizing the Squeeze

f(x) is trapped—it has no choice but to approach L

The Sandwich Metaphor

Think of two slices of bread pressing together.

Upper bound h(x) squeezes f(x) against Lower bound g(x)

The filling goes where the bread goes!

What We Need

  • Find bounding functions g(x) ≤ f(x) ≤ h(x)
  • Bounds must hold near a (not necessarily at a)
  • Both bounds must have the same limit L
  • Then f(x) must also have limit L

Use Squeeze Theorem When...

  • Direct substitution fails
  • Factoring/rationalizing don't help
  • The function oscillates or is bounded
  • You can identify natural upper and lower bounds

Example 1: Classic Squeeze

\[\lim_{x \to 0} x^2 \sin\left(\frac{1}{x}\right)\]
Key observation: sin(1/x) oscillates wildly as x → 0, but it's always bounded!

Step 1: Find Bounds

1
We know: \(-1 \leq \sin\left(\frac{1}{x}\right) \leq 1\)
2
Multiply by \(x^2\) (always positive):
!
\(-x^2\) \(\leq\) \(x^2 \sin\left(\frac{1}{x}\right)\) \(\leq\) \(x^2\)

Visualizing the Squeeze

The oscillating function is trapped between the parabolas

Step 2: Apply the Theorem

1
\(\displaystyle\lim_{x \to 0} (-x^2) = 0\)
2
\(\displaystyle\lim_{x \to 0} x^2 = 0\)
3
Both bounds → 0
✓
By Squeeze Theorem: \(\displaystyle\lim_{x \to 0} x^2 \sin\left(\frac{1}{x}\right) = 0\)
\[\lim_{x \to 0} \frac{\sin x}{x} = 1\]

One of the most important limits in calculus

Geometric Proof

Compare areas in the unit circle

Comparing Areas

For 0 < x < π/2:

Triangle ≤ Sector ≤ Larger Triangle
\[\frac{1}{2}\sin x \leq \frac{1}{2}x \leq \frac{1}{2}\tan x\]

Setting Up the Squeeze

1
From: \(\frac{1}{2}\sin x \leq \frac{1}{2}x \leq \frac{1}{2}\tan x\)
2
Divide by \(\frac{1}{2}\sin x\): \(1 \leq \frac{x}{\sin x} \leq \frac{1}{\cos x}\)
3
Take reciprocals (flip inequalities):
!
\(\cos x\) \(\leq\) \(\frac{\sin x}{x}\) \(\leq\) \(1\)

Completing the Proof

1
\(\displaystyle\lim_{x \to 0} \cos x = 1\)
2
\(\displaystyle\lim_{x \to 0} 1 = 1\)
✓
By Squeeze Theorem: \(\displaystyle\lim_{x \to 0} \frac{\sin x}{x} = 1\)

Important Trig Limits

\(\displaystyle\lim_{x \to 0} \frac{\sin x}{x}\)
= 1
\(\displaystyle\lim_{x \to 0} \frac{1 - \cos x}{x}\)
= 0
\(\displaystyle\lim_{x \to 0} \frac{\tan x}{x}\)
= 1

Memorize these—they appear frequently in calculus

Example 2: Variations

\[\lim_{x \to 0} \frac{\sin 3x}{x}\]
1
Rewrite: \(\frac{\sin 3x}{x} = \frac{\sin 3x}{x} \cdot \frac{3}{3} = 3 \cdot \frac{\sin 3x}{3x}\)
2
As x → 0, we have 3x → 0
✓
\(3 \cdot 1 = 3\)

Example 3

\[\lim_{x \to 0} \frac{\tan x}{x}\]
1
Rewrite: \(\frac{\tan x}{x} = \frac{\sin x}{x} \cdot \frac{1}{\cos x}\)
2
Apply limits: \(1 \cdot \frac{1}{1}\)
✓
= 1

Example 4: Direct Squeeze

\[\lim_{x \to 0} x \cos\left(\frac{1}{x}\right)\]
1
Bound: \(-1 \leq \cos\left(\frac{1}{x}\right) \leq 1\)
2
Therefore: \(-|x| \leq x \cos\left(\frac{1}{x}\right) \leq |x|\)
3
Both \(-|x|\) and \(|x|\) approach 0 as x → 0
✓
= 0

Your Turn

Evaluate:

\[\lim_{x \to 0} \frac{\sin 5x}{2x}\]

Answer: \(\frac{5}{2}\)

Key Takeaways

  • Squeeze Theorem: trap f between bounds that share a limit
  • Works for oscillating or bounded functions
  • \(\frac{\sin x}{x} \to 1\) as x → 0
  • Look for natural bounds like -1 ≤ sin ≤ 1
