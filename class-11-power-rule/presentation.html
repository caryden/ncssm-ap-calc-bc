Unit 2: Differentiation Fundamentals

The Power Rule

Your First Shortcut for Derivatives

Class 11 | CED 2.5, 2.6 | NCSSM

Using the limit definition every time is tedious.

There must be a pattern we can exploit...

Let's Look at What We Know

\(f(x)\) \(f'(x)\)
\(x^1\) \(1\)
\(x^2\) \(2x\)
\(x^3\) \(3x^2\)
\(x^4\) ?

Do you see the pattern?

The exponent comes down as a coefficient...

...and the exponent decreases by 1.

The Power Rule

Power Rule
\[\frac{d}{dx}[x^n] = n \cdot x^{n-1}\]
Works for ANY real number \(n\)
"Bring Down and Reduce"
Bring down the exponent, reduce it by one

Visualizing the Power Rule

The derivative tells us the slope at each point

Power Rule: Positive Integers

\(\frac{d}{dx}[x^2] = 2x\)
\(\frac{d}{dx}[x^3] = 3x^2\)
\(\frac{d}{dx}[x^5] = 5x^4\)
\(\frac{d}{dx}[x^{10}] = 10x^9\)

No limits needed!

Special Case: \(n = 1\)

\[\frac{d}{dx}[x] = \frac{d}{dx}[x^1] = 1 \cdot x^0 = 1\]

The derivative of \(x\) is always \(1\).

The slope of \(y = x\) is 1 everywhere - makes sense!

The Constant Rule

Constant Rule
\[\frac{d}{dx}[c] = 0\]
The derivative of any constant is zero

A horizontal line has zero slope everywhere

Constant Rule Examples

\(\frac{d}{dx}[5] = 0\)
\(\frac{d}{dx}[\pi] = 0\)
\(\frac{d}{dx}[-17] = 0\)
\(\frac{d}{dx}[\sqrt{2}] = 0\)

If there's no \(x\), the derivative is 0!

The Constant Multiple Rule

Constant Multiple Rule
\[\frac{d}{dx}[c \cdot f(x)] = c \cdot f'(x)\]
Constants "factor out" of the derivative

You can pull constants out, differentiate, then put them back.

Constant Multiple Examples

\(\frac{d}{dx}[3x^2] = 3 \cdot 2x = 6x\)
\(\frac{d}{dx}[5x^4] = 5 \cdot 4x^3 = 20x^3\)
\(\frac{d}{dx}[-2x^7] = -2 \cdot 7x^6 = -14x^6\)

Power Rule: Negative Exponents

Remember: \(\frac{1}{x^n} = x^{-n}\)

\(\frac{d}{dx}\left[\frac{1}{x}\right] = \frac{d}{dx}[x^{-1}] = -1 \cdot x^{-2} = -\frac{1}{x^2}\)
\(\frac{d}{dx}\left[\frac{1}{x^3}\right] = \frac{d}{dx}[x^{-3}] = -3x^{-4} = -\frac{3}{x^4}\)

Power Rule: Fractional Exponents

Remember: \(\sqrt{x} = x^{1/2}\) and \(\sqrt[n]{x} = x^{1/n}\)

\(\frac{d}{dx}[\sqrt{x}] = \frac{d}{dx}[x^{1/2}] = \frac{1}{2}x^{-1/2} = \frac{1}{2\sqrt{x}}\)
\(\frac{d}{dx}[\sqrt[3]{x}] = \frac{d}{dx}[x^{1/3}] = \frac{1}{3}x^{-2/3} = \frac{1}{3\sqrt[3]{x^2}}\)

Essential Conversions

\(\sqrt{x}\) = \(x^{1/2}\)
\(\sqrt[3]{x}\) = \(x^{1/3}\)
\(\frac{1}{x}\) = \(x^{-1}\)
\(\frac{1}{x^2}\) = \(x^{-2}\)
\(\sqrt{x^3}\) = \(x^{3/2}\)
\(\frac{1}{\sqrt{x}}\) = \(x^{-1/2}\)

Always convert to exponent form before differentiating!

Sum and Difference Rules

Sum Rule
\[\frac{d}{dx}[f(x) + g(x)] = f'(x) + g'(x)\]
Difference Rule
\[\frac{d}{dx}[f(x) - g(x)] = f'(x) - g'(x)\]

Differentiate term by term!

Visualizing the Sum Rule

The slope of the sum equals the sum of the slopes

Example: Polynomial

Find \(\frac{d}{dx}[4x^3 - 2x^2 + 5x - 7]\)
Apply rules to each term:
\(\frac{d}{dx}[4x^3] - \frac{d}{dx}[2x^2] + \frac{d}{dx}[5x] - \frac{d}{dx}[7]\)
\(= 4(3x^2) - 2(2x) + 5(1) - 0\)
\(= 12x^2 - 4x + 5\)

Example: Rewrite First

Find \(\frac{d}{dx}\left[\frac{3x^4 + 2}{x^2}\right]\)
Rewrite: \(\frac{3x^4}{x^2} + \frac{2}{x^2} = 3x^2 + 2x^{-2}\)
Differentiate: \(\frac{d}{dx}[3x^2] + \frac{d}{dx}[2x^{-2}]\)
\(= 6x + 2(-2)x^{-3}\)
\(= 6x - \frac{4}{x^3}\)

Common Mistake!

The Power Rule is for \(x^n\), NOT \(a^x\)

\(\frac{d}{dx}[2^x] \neq x \cdot 2^{x-1}\)

Power Rule applies to:

\(x^5\), \(x^{-2}\), \(x^{1/3}\)

Variable base, constant exponent

Power Rule does NOT apply to:

\(2^x\), \(e^x\), \(10^x\)

Constant base, variable exponent

All Rules Together

1
Constant: \(\frac{d}{dx}[c] = 0\)
2
Power: \(\frac{d}{dx}[x^n] = nx^{n-1}\)
3
Constant Multiple: \(\frac{d}{dx}[cf(x)] = c \cdot f'(x)\)
4
Sum/Difference: Differentiate term by term

Practice: Find \(f'(x)\)

\(f(x) = 5x^4 - 3x^2 + 2x - 8\)

1 minute

Solution

\(f(x) = 5x^4 - 3x^2 + 2x - 8\)
\(f'(x) = 5(4x^3) - 3(2x) + 2(1) - 0\)
\(f'(x) = 20x^3 - 6x + 2\)

Practice: Find \(g'(x)\)

\(g(x) = \sqrt{x} + \frac{4}{x}\)

Hint: Rewrite as exponents first!

2 minutes

Solution

\(g(x) = \sqrt{x} + \frac{4}{x} = x^{1/2} + 4x^{-1}\)
\(g'(x) = \frac{1}{2}x^{-1/2} + 4(-1)x^{-2}\)
\(g'(x) = \frac{1}{2}x^{-1/2} - 4x^{-2}\)
\(g'(x) = \frac{1}{2\sqrt{x}} - \frac{4}{x^2}\)

Key Takeaways

  • Power Rule: \(\frac{d}{dx}[x^n] = nx^{n-1}\) for any real \(n\)
  • Constants vanish: \(\frac{d}{dx}[c] = 0\)
  • Constants factor out: \(\frac{d}{dx}[cf] = c \cdot f'\)
  • Term by term: Sums and differences differentiate separately
  • Rewrite first: Convert radicals and fractions to exponents
