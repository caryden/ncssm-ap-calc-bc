Unit 6: Integration and Accumulation of Change BC Only

Integration by Parts

The Product Rule in Reverse

Class 36 | CED 6.11 | NCSSM

Substitution Doesn't Work Here

\(\displaystyle \int xe^x\,dx\)

• If \(u = x\), then \(du = dx\), but \(e^x\,dx\) is not just \(du\)

• If \(u = e^x\), then \(du = e^x\,dx\), but we can't eliminate the \(x\)

Neither substitution simplifies this integral

U-substitution is the chain rule backwards.

What's the product rule backwards?

From Product Rule to Integration by Parts

Product rule: \(\displaystyle \frac{d}{dx}[uv] = u\frac{dv}{dx} + v\frac{du}{dx}\)

Multiply by \(dx\): \(d(uv) = u\,dv + v\,du\)

Rearrange: \(u\,dv = d(uv) - v\,du\)

Integrate both sides...

Integration by Parts

\(\displaystyle \int u\,dv = uv - \int v\,du\)

We trade one integral for another — hopefully easier!

The whole point is to replace an integral we can't do with one we can

Success means the new integral \(\int v\,du\) is simpler than the original \(\int u\,dv\)

What Do We Choose?

You choose \(u\) (what to differentiate) and \(dv\) (what to integrate)

Then you COMPUTE \(du\) (by differentiating \(u\)) and \(v\) (by integrating \(dv\))

Warning: You pick \(u\) and \(dv\), NOT \(u\) and \(v\) directly!

LIATE: A Guideline for Choosing u

L Logarithmic \(\ln x\), \(\log x\)
I Inverse Trig \(\arctan x\), \(\arcsin x\)
A Algebraic \(x\), \(x^2\), polynomials
T Trigonometric \(\sin x\), \(\cos x\)
E Exponential \(e^x\), \(2^x\)

Choose u from highest priority present; rest goes to dv

The Logic Behind LIATE

  • L and I: become simpler when differentiated (good for u)
  • A: polynomials reduce degree when differentiated
  • T and E: stay manageable when integrated (good for dv)

LIATE is a GUIDELINE, not a rule. The goal is making \(\int v\,du\) easier.

Example 1: \(\int xe^x\,dx\)

LIATE: x is Algebraic, e^x is Exponential. A > E, so u = x

\(u = x\)
\(dv = e^x\,dx\)
\(du = dx\)
\(v = e^x\)

\(\int xe^x\,dx = xe^x - \int e^x\,dx\)

\(= xe^x - e^x + C = e^x(x-1) + C\)

Example 2: \(\int \ln x\,dx\)

Rewrite: \(\int 1 \cdot \ln x\,dx\). LIATE: ln x is Logarithmic (highest)

\(u = \ln x\)
\(dv = dx\)
\(du = \frac{1}{x}dx\)
\(v = x\)

\(\int \ln x\,dx = x\ln x - \int x \cdot \frac{1}{x}\,dx\)

\(= x\ln x - x + C\)

Example 3: \(\int x^2\cos x\,dx\) (Two Applications)

First IBP: \(u = x^2\), \(dv = \cos x\,dx\)

\(= x^2\sin x - \int 2x\sin x\,dx\)

Second IBP on remaining: \(u = 2x\), \(dv = \sin x\,dx\)

\(= x^2\sin x - [-2x\cos x - \int -2\cos x\,dx]\)

\(= x^2\sin x + 2x\cos x - 2\sin x + C\)

Some integrals need IBP applied more than once

Polynomial × trig or polynomial × exponential often needs IBP until the polynomial disappears

Example 4: Cyclic — \(\int e^x\cos x\,dx\)

First IBP: \(u = e^x\), \(dv = \cos x\,dx\)

\(= e^x\sin x - \int e^x\sin x\,dx\)

Second IBP: \(u = e^x\), \(dv = \sin x\,dx\)

\(= e^x\sin x - [-e^x\cos x - \int -e^x\cos x\,dx]\)

\(= e^x\sin x + e^x\cos x - \int e^x\cos x\,dx\)

We got back to the original integral!

Solving Algebraically

Let \(I = \int e^x\cos x\,dx\)

We found: \(I = e^x\sin x + e^x\cos x - I\)

Solve: \(2I = e^x(\sin x + \cos x)\)

\(\displaystyle I = \frac{e^x(\sin x + \cos x)}{2} + C\)

This technique is key for AP success!

The Tabular (DI) Method

For repeated IBP (polynomial × exp/trig), use a table:

\(\int x^3e^x\,dx\)

D (derivatives) I (integrals) Sign
\(x^3\)\(e^x\)+
\(3x^2\)\(e^x\)−
\(6x\)\(e^x\)+
\(6\)\(e^x\)−
\(0\)\(e^x\)+

Reading the Table

Draw diagonal arrows, multiply, use alternating signs:

\(x^3e^x - 3x^2e^x + 6xe^x - 6e^x + C\)

Much faster than writing out multiple IBP applications!

Choosing Your Approach

Standard IBP

  • • One or two applications
  • • Non-polynomial factors
  • • Show all steps

Tabular Method

  • • Polynomial × exp or trig
  • • Many applications needed
  • • Efficient computation

Algebraic Solving

  • • Cyclic integrals
  • • Returns to original
  • • e^x × trig patterns

Avoid These Errors

  • ✗ DON'T choose u and v directly — you choose u and dv
  • ✗ DON'T forget the MINUS sign: \(uv - \int v\,du\)
  • ✗ DON'T stop too early — some need multiple applications
  • ✗ DON'T give up on cyclic — solve algebraically!
  • ✗ DON'T choose dv that you can't integrate

Practice 1

\(\displaystyle \int x\sin x\,dx\)

Identify u and dv using LIATE

Time: 3 minutes

Solution

\(u = x\)
\(dv = \sin x\,dx\)
\(du = dx\)
\(v = -\cos x\)

\(\int x\sin x\,dx = -x\cos x - \int -\cos x\,dx\)

\(= -x\cos x + \sin x + C\)

Practice 2: Setup Only

Set up (don't fully evaluate) IBP for:

\(\displaystyle \int x^2e^{2x}\,dx\)

• Identify u, dv, du, v

• Write the first application

• Note how many total applications needed

Solution

\(u = x^2\)
\(dv = e^{2x}\,dx\)
\(du = 2x\,dx\)
\(v = \frac{1}{2}e^{2x}\)

First step: \(\displaystyle \frac{x^2e^{2x}}{2} - \int xe^{2x}\,dx\)

Needs two total applications (polynomial degree 2)

Key Takeaways

  • Integration by parts: \(\int u\,dv = uv - \int v\,du\)
  • LIATE helps choose u: Logs, Inverse trig, Algebraic, Trig, Exp
  • Choose u and dv, then COMPUTE du and v
  • Some integrals need multiple applications
  • Cyclic integrals: solve algebraically when you return to the original

Common Integrals Using IBP

  • \(\int xe^x\,dx = e^x(x-1) + C\)
  • \(\int x\sin x\,dx = -x\cos x + \sin x + C\)
  • \(\int x\cos x\,dx = x\sin x + \cos x + C\)
  • \(\int \ln x\,dx = x\ln x - x + C\)
  • \(\int e^x\sin x\,dx = \frac{e^x(\sin x - \cos x)}{2} + C\)
