Partial Fractions

Un-Adding Fractions to Integrate

Class 37 | CED 6.12 | NCSSM

How Do We Integrate This?

\[\int \frac{2x+3}{x^2+3x+2}\,dx\]

  • U-substitution doesn't work (numerator isn't derivative of denominator)
  • Integration by parts doesn't fit
  • What should we do?

What If We Could Simplify?

Watch what happens when we add fractions:

\[\frac{1}{x+1} + \frac{1}{x+2} = \frac{(x+2)+(x+1)}{(x+1)(x+2)} = \frac{2x+3}{x^2+3x+2}\]

↑

That's our integrand! Can we reverse this process?

Partial fractions reverses fraction addition: we break a complex fraction into simpler pieces

Each simple piece is easy to integrate!

Compare the Integrals

Original

\[\int \frac{2x+3}{x^2+3x+2}\,dx\]

Not obvious...

Decomposed

\[\int \frac{1}{x+1}\,dx + \int \frac{1}{x+2}\,dx\]

\[= \ln|x+1| + \ln|x+2| + C\]

Easy!

The Building Block

\[\int \frac{A}{x-a}\,dx = A\ln|x-a| + C\]

Every partial fraction term integrates to a logarithm

(or a power rule when the exponent is higher)

Before Decomposing

  • Must be PROPER: degree(numerator) < degree(denominator)
    If not, do polynomial long division first
  • Denominator must be FACTORED completely
    Into linear factors
AP BC focuses on linear factors only

Distinct Linear Factors

Template

\[\frac{P(x)}{(x-a)(x-b)} = \frac{A}{x-a} + \frac{B}{x-b}\]

Example

\[\frac{2x+3}{(x+1)(x+2)} = \frac{A}{x+1} + \frac{B}{x+2}\]

One term per distinct factor

Repeated Linear Factors

For \((x-a)^2\)

\[\frac{P(x)}{(x-a)^2} = \frac{A}{x-a} + \frac{B}{(x-a)^2}\]

For \((x-a)^3\)

\[\frac{P(x)}{(x-a)^3} = \frac{A}{x-a} + \frac{B}{(x-a)^2} + \frac{C}{(x-a)^3}\]

Factor \((x-a)^n\) needs n terms with powers 1 through n

Mixed Factors

For \((x-a)(x-b)^2\)

\[\frac{P(x)}{(x-a)(x-b)^2} = \frac{A}{x-a} + \frac{B}{x-b} + \frac{C}{(x-b)^2}\]

Distinct factor: one term   |   Repeated factor: multiple terms

Method 1: Strategic Substitution

Step 1 Multiply both sides by the common denominator
Step 2 Substitute x-values that make factors zero
Step 3 Each substitution gives one constant directly
Fast and efficient for most cases

Example 1: \(\int \frac{1}{x^2-1}\,dx\)

Factor \(x^2 - 1 = (x-1)(x+1)\)
Decompose \(\displaystyle\frac{1}{(x-1)(x+1)} = \frac{A}{x-1} + \frac{B}{x+1}\)
Clear \(1 = A(x+1) + B(x-1)\)

Example 1 Continued

From: \(1 = A(x+1) + B(x-1)\)

x = 1 \(1 = A(2) + B(0) \Rightarrow A = \frac{1}{2}\)
x = -1 \(1 = A(0) + B(-2) \Rightarrow B = -\frac{1}{2}\)

\[\frac{1}{x^2-1} = \frac{1/2}{x-1} - \frac{1/2}{x+1}\]

Example 1 Completed

Integrate \(\displaystyle\int \frac{1}{x^2-1}\,dx = \int \frac{1/2}{x-1}\,dx - \int \frac{1/2}{x+1}\,dx\)
Result \(\displaystyle= \frac{1}{2}\ln|x-1| - \frac{1}{2}\ln|x+1| + C\)
Simplify \(\displaystyle= \frac{1}{2}\ln\left|\frac{x-1}{x+1}\right| + C\)

Example 2: \(\int \frac{5x-4}{(x-2)(x+1)}\,dx\)

Decompose \(\displaystyle\frac{5x-4}{(x-2)(x+1)} = \frac{A}{x-2} + \frac{B}{x+1}\)
Clear \(5x - 4 = A(x+1) + B(x-2)\)
x = 2 \(6 = 3A \Rightarrow A = 2\)
x = -1 \(-9 = -3B \Rightarrow B = 3\)
Integrate \(2\ln|x-2| + 3\ln|x+1| + C\)

Example 3: \(\int \frac{x+3}{x^2(x+1)}\,dx\)

Setup \(\displaystyle\frac{x+3}{x^2(x+1)} = \frac{A}{x} + \frac{B}{x^2} + \frac{C}{x+1}\)
\(x^2\) is a repeated factor, so we need both \(\frac{A}{x}\) and \(\frac{B}{x^2}\)

Example 3 Continued

Clear: \(x + 3 = Ax(x+1) + B(x+1) + Cx^2\)

x = 0 \(3 = B(1) \Rightarrow B = 3\)
x = -1 \(2 = C(1) \Rightarrow C = 2\)
Match \(x^2\) \(0 = A + C \Rightarrow A = -2\)

Example 3 Completed

Integrate \(\displaystyle\int \left(\frac{-2}{x} + \frac{3}{x^2} + \frac{2}{x+1}\right)\,dx\)
Result \(\displaystyle= -2\ln|x| - \frac{3}{x} + 2\ln|x+1| + C\)
Note: \(\displaystyle\int \frac{3}{x^2}\,dx = -\frac{3}{x} + C\) (power rule!)

What If It's Improper?

If degree(numerator) ≥ degree(denominator):

Do long division FIRST!

Result: quotient + proper fraction

Then decompose the proper part

Example 4: \(\int \frac{x^2+1}{x^2-1}\,dx\)

Check degree(num) = 2 = degree(denom). Improper!
Divide \(\displaystyle\frac{x^2+1}{x^2-1} = 1 + \frac{2}{x^2-1}\)
Integrate \(\displaystyle\int 1\,dx + \int \frac{2}{x^2-1}\,dx\)
Result \(= x + \ln|x-1| - \ln|x+1| + C\)

Method 2: Coefficient Matching

Alternative approach:

Step 1 Clear denominators
Step 2 Expand the right side completely
Step 3 Match coefficients of like powers of x
Step 4 Solve the resulting system of equations
More algebraically tedious but always works

Always verify your decomposition by recombining the fractions

Or differentiate your final answer to check

Practice 1

Set up the partial fraction decomposition for:

\[\frac{3x}{(x-1)(x+2)^2}\]

Don't solve - just write the correct form

Time: 1 minute

Solution

\[\frac{3x}{(x-1)(x+2)^2} = \frac{A}{x-1} + \frac{B}{x+2} + \frac{C}{(x+2)^2}\]

\((x+2)^2\) is repeated, so it needs TWO terms

Practice 2

Evaluate:

\[\int \frac{2x+5}{x^2-4}\,dx\]

Time: 4 minutes

Solution

Factor \(x^2 - 4 = (x-2)(x+2)\)
Decompose \(\displaystyle\frac{A}{x-2} + \frac{B}{x+2}\); clear: \(2x + 5 = A(x+2) + B(x-2)\)
x = 2 \(9 = 4A \Rightarrow A = \frac{9}{4}\)
x = -2 \(1 = -4B \Rightarrow B = -\frac{1}{4}\)
Result \(\displaystyle\frac{9}{4}\ln|x-2| - \frac{1}{4}\ln|x+2| + C\)

Key Takeaways

  • Partial fractions breaks rational functions into simpler pieces
  • Factor the denominator completely first
  • Distinct factor: one term. Repeated factor \((x-a)^n\): n terms
  • Find constants by strategic substitution or coefficient matching
  • Each term integrates to a logarithm (or power)

Decomposition Quick Reference

Denominator Decomposition
\((x-a)(x-b)\) \(\frac{A}{x-a} + \frac{B}{x-b}\)
\((x-a)^2\) \(\frac{A}{x-a} + \frac{B}{(x-a)^2}\)
\((x-a)(x-b)^2\) \(\frac{A}{x-a} + \frac{B}{x-b} + \frac{C}{(x-b)^2}\)
