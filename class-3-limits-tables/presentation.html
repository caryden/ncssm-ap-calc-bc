From Pictures to Numbers

Yesterday we traced curves. Today we read numbers.

Watch the pattern in function values as x approaches a.

The Table Method

  • Choose x-values approaching from the LEFT
  • Choose x-values approaching from the RIGHT
  • Calculate f(x) for each value
  • Look for the pattern—what value do we approach?

Getting Closer by Factors of 10

Add 9s to get closer: 1.9 → 1.99 → 1.999 → 1.9999

Building the Table: \(f(x) = \frac{x^2 - 4}{x - 2}\)

x f(x)

Click the buttons to build your table. What pattern do you see?

Reading the Table

x (from left) f(x) x (from right) f(x)
1.9 3.9 2.1 4.1
1.99 3.99 2.01 4.01
1.999 3.999 → 2.001 4.001
1.9999 3.9999 4 2.0001 4.0001
\(\lim_{x \to 2} \frac{x^2 - 4}{x - 2} = 4\)

Approaching from Both Sides

From Left (x < 2)

x = 1.9, 1.99, 1.999, ...

f(x) → 4

From Right (x > 2)

x = 2.1, 2.01, 2.001, ...

f(x) → 4

Both sides must agree for the limit to exist!

Table-Building Checklist

  • At least 3-4 values from each side
  • Values getting progressively closer (by factors of 10)
  • Never include x = a itself
  • Check that both sides tell the same story

Case 1: Limit Exists

Both sides converge to the same value

What Tables Can Tell Us

Limit Exists

Both sides → same value

Limit exists

One-Sided Differ

Left ≠ Right

Limit DNE

Unbounded

Values → ±∞

Limit DNE

Oscillation

No pattern emerges

Limit DNE

Tables suggest limits. They don't prove them.

For certainty, we need algebraic methods.

Using Your Calculator (TI-84)

Step 1: Press Y= and enter your function

Step 2: Press 2nd → TABLE

Step 3: Use 2nd → TBLSET to set:

  • TblStart = your starting x-value
  • ΔTbl = your step size (e.g., 0.01)

Using Desmos Tables

Step 1: Click the + button, select "Table"

Step 2: In the first column, type your x-values

Step 3: Click the column header and enter your function

Tip: Desmos will calculate f(x) automatically!

Beware of Calculator Precision

Try: \(\frac{\sin(x)}{x}\) for very small x

xsin(x)/xNotes

0.010.99998...Good

Solutions

Problem 1

\(\lim_{x \to 3} \frac{x^2 - 9}{x - 3}\)

Table shows: 5.9, 5.99, 5.999... and 6.1, 6.01, 6.001...

Limit = 6

Problem 2

\(\lim_{x \to 0} \frac{|x|}{x}\)

Left: -1, -1, -1...
Right: 1, 1, 1...

Limit DNE (left ≠ right)

Key Takeaways

  • Tables reveal limit behavior through patterns
  • Approach from both sides with values getting closer
  • Both sides must agree for the limit to exist
  • Tables suggest; algebra proves
